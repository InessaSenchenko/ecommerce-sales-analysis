# E-Commerce Sales Analysis

## Опис проєкту
Аналіз продажів британського інтернет-магазину подарунків та декору.
541 909 транзакцій за 2010-2011 роки.

## Інструменти
- Python (Pandas)
- Tableau Public

## Що зроблено
- Очищення та підготовка даних
- Розрахунок ключових метрик: виручка £8.9M, AOV £480
- RFM сегментація 4 338 клієнтів
- Інтерактивний дашборд

## Ключові висновки
- Сезонний пік продажів у вересні–листопаді (+93%)
- 34.7% клієнтів зробили лише одне замовлення
- 2 153 клієнти у зоні ризику відтоку
